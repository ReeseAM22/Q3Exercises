HTML

HyperText Markup Language

By using HTML, you can create your own websites. This is the main structure that tells how certain elements show up. These elements are called tags and they are denoted with these angled brackets, < >.

CSS

Cascading Style Sheet

If HTML is the structure of the website, CSS is the layout. This can show designs or more commonly known as style. Examples: Fonts, spacings, colors, and images.

Bootstrap

Bootstrap Version 4

Here we have the Bootstrap, this time they are the framework. Their main objective is to help make an easier way for web pages to fit to different devices. That is called Responsive Web Design.
